# node-hapi-starter

## Tech Stack
- hapijs - a rich framework for building applications and services
- hapi-mrhorse - to create policies like allow access to rest api only if logged in
- hapi-swagger
- hapi-server-session

## Running on local
- npm install
- nodemon .
